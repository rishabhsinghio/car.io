This is a ready made car with physics and touch controls. For Mobile and Tablets.
Below are the some steps to get started.

**How to get started**
To go Godot Engine> Project> Project Settings> Input Maps
then add input action buttons "Brakes", "Throttle" and "Cam Switch" to start immediatly. You can add other input buttons also.

**To get correct resolution in mobile/tablets:**
Project> Project Settings> General> Display> Window
then set 
Mode: Canvas Item
Aspect: Expand
